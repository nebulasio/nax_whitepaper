\section{NAX分析}
\subsection{发行量分析}
由于每期增发池中未发放的部分滚入下一期增发池中，因此总发行量为固定值，可以因此推导出初始的预增发量，如果我们将总发行量设定为100亿（\(10^{10}\)），那么可以推导出第0期预发行的规模如下：
\begin{equation}
  \sum_{i,j} K_{i,j} = \sum_i C_0 \mu^i = \frac{C_0}{1-\mu}
\end{equation}
  令此上界为100亿（\(10^{10}\)），可解出\(C_0 = 10^{10}(1-\mu) = 1.0\times10^7\)。
